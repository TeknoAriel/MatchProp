# MatchProp — Foco 2026 (producto y prioridades)

Documento de **re-enfoque**: qué impulsamos ahora y qué dejamos explícitamente en segundo plano.  
**No reemplaza** al [masterplan.md](./masterplan.md) (fuente de verdad); lo **operacionaliza** para decisiones del trimestre.

**Última actualización:** 2026-03-20

---

## North star (una frase)

**MatchProp** es la plataforma donde el comprador descubre propiedades (feed + lista + búsqueda activa + alertas) y las inmobiliarias reciben **leads bajo funnel anti-cierre** (PENDING → ACTIVE) e integración con **Kiteprop / CRM** (reverse-matching, outbox).

---

## En foco (prioridad alta)

| Área                          | Por qué                                                                                                                                          |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Estabilidad de plataforma** | CI verde, deploy predecible, URLs canónicas (web / API / admin), ingest y cron documentados.                                                     |
| **Core B2C**                  | Feed, lista, swipe, búsqueda activa global, alertas, asistente (parser + UX contract del masterplan).                                            |
| **Core B2B / integración**    | Leads, activación, CRM push / outbox, importadores Kiteprop con datos reales en prod.                                                            |
| **Observabilidad mínima**     | Health, logs sin PII, smoke prod; documentación operativa al día ([PROD.md](./PROD.md), [ESTABILIDAD_Y_RELEASE.md](./ESTABILIDAD_Y_RELEASE.md)). |
